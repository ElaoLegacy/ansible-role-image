Ansible Role - Image
====================

A image role to install image manipulation tools on elao symfony standard vagrant box


Requirements
------------

This role only run on elao symfony standard vagrant box. See https://vagrantcloud.com/elao/symfony-standard-debian


Example Playbook
----------------

    - hosts: servers
      roles:
         - { role: elao.image }
